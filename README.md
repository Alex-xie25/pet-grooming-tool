# Pet Grooming Tool

A web-based interactive tool to help pet owners configure grooming options for their pets. Built with React and hosted on Cloudflare Pages.  

This tool is ideal for showcasing your pet grooming product options, collecting user preferences, and can be embedded into websites, including WordPress pages.

Learn more: [KUDI](https://thekudipet.com/about/)
---

## **Features**

- Interactive grooming configuration tool
- Fully responsive: works on desktop and mobile
- Can be embedded via iframe in WordPress or other websites
- Easy to deploy using Cloudflare Pages
- Collect user preferences for further engagement
